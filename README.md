# CUFE-DataStructure2024

中央财经大学 2024 年数据结构课程相关代码示例与期末复习资料。本项目主要面向正在学习数据结构和 C 语言实现的同学，便于按章节阅读、编译、运行和复习。

## 项目内容

| 目录/文件 | 内容 |
| --- | --- |
| `链表/` | 单链表、双链表、循环链表的创建、插入、删除、查找和逆置 |
| `栈/` | 顺序栈、两个栈共享数组空间 |
| `队列/` | 循环队列的基本操作 |
| `串/` | 顺序串、链式串、字符串连接、回文判断 |
| `查找/` | 二分查找及递归二分查找 |
| `排序/` | 冒泡排序、选择排序、插入排序、希尔排序 |
| `期末复习.md` / `期末复习.pdf` | 课程复习笔记 |

## 推荐学习顺序

1. `链表/`：先理解指针、节点、头插法、尾插法和常见删除操作。
2. `栈/` 与 `队列/`：掌握受限线性表的顺序存储实现。
3. `串/`：练习字符数组、动态存储和字符串遍历。
4. `查找/`：理解有序表上的二分查找思想。
5. `排序/`：比较基础排序算法的过程、复杂度和适用场景。
6. `期末复习.md`：结合代码回顾常见题型和实现模板。

## 环境准备

需要安装任意 C 语言编译器，例如：

- Windows: MinGW / Dev-C++ 自带 GCC / MSYS2 GCC
- macOS: Xcode Command Line Tools
- Linux: GCC

确认 `gcc` 可用：

```sh
gcc --version
```

## 编译和运行单个示例

进入对应目录后编译，例如：

```sh
cd 链表
gcc -std=c17 -Wall -Wextra -pedantic 单链表逆置.c -o 单链表逆置
./单链表逆置
```

在 Windows PowerShell 中，生成的可执行文件通常带 `.exe` 后缀：

```powershell
cd 链表
gcc -std=c17 -Wall -Wextra -pedantic 单链表逆置.c -o 单链表逆置.exe
.\单链表逆置.exe
```

## 批量编译检查

仓库提供了 `build.ps1`，用于批量编译所有 `.c` 示例，适合检查代码是否都能通过编译。

```powershell
.\build.ps1
```

默认输出目录为 `build/`。如果想指定输出目录：

```powershell
.\build.ps1 -OutputDir .\out
```

## 学习建议

- 先运行代码，再阅读实现；运行结果能帮助理解指针变化和数组下标变化。
- 对链表、栈、队列类代码，重点观察空结构、满结构、头节点、尾节点等边界情况。
- 对排序和查找代码，建议手写一遍数组变化过程，再对照程序运行结果。
- 修改示例输入后重新编译运行，检查自己是否真正理解了算法流程。
