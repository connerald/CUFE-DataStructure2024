# 串

本目录包含字符串和串结构的相关示例。学习重点是字符数组、字符串结束标记、顺序存储和链式存储。

## 文件说明

| 文件 | 内容 |
| --- | --- |
| `顺序表相关操作.c` | 顺序存储串的基础操作 |
| `链式串相关操作.c` | 链式存储串的基础操作 |
| `判断回文字符串.c` | 判断字符串是否为回文 |
| `不使用strcat实现连接字符串s和t.c` | 手动实现字符串连接 |

## 复习重点

- C 字符串以 `\0` 作为结束标记。
- 字符数组需要预留结束标记空间。
- 字符串遍历时要注意下标边界。
- 手写字符串函数时，重点检查目标数组容量是否足够。
