# 排序

本目录包含基础内部排序算法示例。建议对照数组变化过程理解每种排序的思想。

## 文件说明

| 文件 | 内容 |
| --- | --- |
| `冒泡排序.c` | 相邻元素比较和交换 |
| `选择排序.c` | 每轮选择未排序部分中的最小值 |
| `插入排序.c` | 将当前元素插入到前面有序部分 |
| `希尔排序.c` | 分组插入排序，逐步缩小增量 |

## 复杂度参考

| 算法 | 平均时间复杂度 | 空间复杂度 | 稳定性 |
| --- | --- | --- | --- |
| 冒泡排序 | `O(n^2)` | `O(1)` | 稳定 |
| 选择排序 | `O(n^2)` | `O(1)` | 不稳定 |
| 插入排序 | `O(n^2)` | `O(1)` | 稳定 |
| 希尔排序 | 与增量序列有关 | `O(1)` | 不稳定 |

## 复习重点

- 冒泡排序是否能提前结束。
- 选择排序每轮确定一个最终位置。
- 插入排序适合基本有序的数据。
- 希尔排序重点理解“分组”和“缩小增量”。
