# 链表

本目录包含单链表、双链表和双向循环链表的基础操作示例。学习重点是理解节点结构、指针连接关系，以及插入、删除、逆置时指针如何变化。

## 文件说明

| 文件 | 内容 |
| --- | --- |
| `建立单链表.c` | 使用尾插法建立单链表，并打印、释放链表 |
| `头插法创建链表.c` | 使用头插法创建链表 |
| `尾插法创建链表.c` | 使用尾插法创建链表 |
| `单链表逆置.c` | 将单链表原地逆置 |
| `双链表查找插入删除.c` | 双链表的查找、插入和删除 |
| `2.11_设计算法删除单链表中值相同的多余节点.c` | 删除重复值节点 |
| `2.18_从单链表中删除值为x的节点（多个）.c` | 删除所有值为 `x` 的节点 |
| `2.23_将双向循环链表逆置.c` | 逆置双向循环链表 |

## 建议顺序

1. `建立单链表.c`
2. `头插法创建链表.c`
3. `尾插法创建链表.c`
4. `单链表逆置.c`
5. 删除类题目
6. 双链表和双向循环链表题目

## 复习重点

- 空链表是否能正确处理。
- 头节点被删除或替换时，头指针是否同步更新。
- 插入和删除后，是否存在断链或内存泄漏。
- `malloc` 后是否初始化节点，使用完成后是否 `free`。
